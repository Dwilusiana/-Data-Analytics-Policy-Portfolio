# Inflation Forecasting (Time Series Analysis)

## Problem Context
Inflation monitoring requires short-term forecasting and an understanding of uncertainty to support timely decision-making.

## Data Source
- Monthly inflation data from Badan Pusat Statistik (BPS)

## Methodology
- Time series decomposition
- Stationarity testing (ADF)
- ARIMA-based forecasting

## Key Outputs
- Short-term inflation forecasts
- Confidence intervals
- Executive summary (available in `reports/`)

## Key Insight
While inflation exhibits short-term volatility, forecasts suggest reversion toward historical ranges, indicating temporary rather than structural pressures.
