# Inflation Monitoring Dashboard

## Objective
Interactive dashboard to monitor monthly inflation trends across regions
and support early identification of inflationary pressure.

## Key Features
- Year-based filtering
- Monthly inflation trend by region
- Average inflation comparison
- Executive-level insights for decision support

## Data Source
Public / synthetic economic data inspired by BPS inflation statistics.

## Tools
- Python
- Streamlit
- Pandas
- Plotly

## Use Case
Designed for analysts or policymakers to monitor short-term inflation dynamics
and identify regions requiring closer observation.
