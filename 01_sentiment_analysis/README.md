# Economic & Financial Sentiment Analysis

## Problem Context
Public perception plays an important role in shaping economic expectations and policy credibility. This project analyzes sentiment in public discourse related to economic issues.

## Data Source
- Online news articles and public media sources
- Publicly accessible and legally obtained data

## Methodology
- Text preprocessing and cleaning
- Baseline sentiment analysis
- Transformer-based sentiment modeling (IndoBERT)

## Key Outputs
- Sentiment distribution and trends
- Comparison between baseline and advanced models
- Interpretative insights for perception monitoring

## Key Insight
Public sentiment exhibits measurable shifts around major economic narratives, highlighting the importance of monitoring perception alongside quantitative indicators.
