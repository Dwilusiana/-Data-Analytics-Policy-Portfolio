##Mini SQL project for exploratory analysis of economic indicators.
## Development Process (SDLC)

1. Requirement Understanding  
   Defined analytical questions based on economic monitoring needs.

2. Data Preparation  
   Structured regional inflation data and ensured consistency.

3. Analysis & Query Development  
   Developed SQL queries for aggregation, ranking, and anomaly detection.

4. Validation  
   Cross-checked results for logical consistency and trend alignment.

5. Insight Delivery  
   Summarized findings into actionable insights for stakeholders.
